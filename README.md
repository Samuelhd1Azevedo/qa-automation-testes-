# Desafio de Automação de Testes (Python)

Este repositório contém a automação de testes para a API da **Petstore Swagger** e para o fluxo funcional E2E da plataforma web **SauceDemo**.

## 🛠️ Tecnologias Utilizadas
- **Linguagem Principal:** Python 3.11+
- **Automação de API:** Requests & Pytest
- **Automação Web:** Selenium WebDriver & Pytest
- **Framework de Testes:** Pytest
- **CI/CD:** GitHub Actions
- **Padrão de Projeto:** Page Objects (módulo Web)

## 🚀 Como Executar o Projeto Localmente

### Pré-requisitos
- Python 3 instalado no sistema.

### Executando testes de API
```bash
cd automacao_api
pip install -r requirements.txt
pytest tests/
